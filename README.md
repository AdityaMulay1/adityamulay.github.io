# 🚀 Portfolio - ADITYA MULAY

A modern web application built using Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## 🛠️ Tech Stack

- **Frontend**: React + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Library**: shadcn-ui
